# Cloud Infrastructure Plugin

Complete cloud infrastructure toolkit for Claude Code with specialized agents and skills for modern DevOps, cloud architecture, and infrastructure as code.

## Agents

### cloud-architect
Strategic cloud architecture expert for designing scalable, secure, and cost-effective cloud solutions across AWS, GCP, and Azure.

### kubernetes-architect
Kubernetes specialist for container orchestration, cluster design, workload management, and cloud-native patterns.

### terraform-specialist
Infrastructure as Code expert for Terraform modules, state management, and multi-environment deployments.

### deployment-engineer
CI/CD and deployment automation expert for building robust pipelines and release strategies.

### network-engineer
Cloud networking specialist for VPCs, subnets, load balancers, DNS, and security groups.

### hybrid-cloud-architect
Multi-cloud and hybrid infrastructure expert for designing solutions spanning on-premises and cloud environments.

### service-mesh-expert
Service mesh specialist for Istio, Linkerd, and microservices communication patterns.

## Skills

### terraform-module-library
Reusable Terraform modules and patterns for AWS, GCP, and Azure infrastructure.

### multi-cloud-architecture
Patterns for designing and deploying across multiple cloud providers.

### cost-optimization
Cloud cost management strategies, reserved instances, spot instances, and resource optimization.

### hybrid-cloud-networking
VPN, Direct Connect, ExpressRoute, and hybrid connectivity patterns.

### istio-traffic-management
Istio configuration for traffic routing, canary deployments, and circuit breakers.

### linkerd-patterns
Linkerd service mesh patterns for mTLS, observability, and traffic management.

### mtls-configuration
Mutual TLS setup and certificate management for secure service-to-service communication.

### service-mesh-observability
Distributed tracing, metrics, and logging for service mesh environments.

## Installation

```bash
/plugin install cloud-infrastructure
```

## Usage Examples

**Cloud architecture design:**
```
Design a multi-region AWS architecture with disaster recovery
```

**Kubernetes deployment:**
```
Help me set up a production-ready EKS cluster with autoscaling
```

**Terraform modules:**
```
Create a Terraform module for an AWS VPC with public and private subnets
```

**Service mesh setup:**
```
Configure Istio for canary deployments with traffic splitting
```
